# Caso 28: Sistema de Calificación con Estrellas

Este proyecto es un componente interactivo que permite a los usuarios calificar con un sistema de 1 a 5 estrellas.

## Funcionalidades
- Interacción visual al pasar el mouse (hover).
- Persistencia de datos usando `localStorage`.
- Cálculo dinámico del promedio de votos.

## Tecnologías usadas
- HTML5
- CSS3
- JavaScript (Vanilla)
